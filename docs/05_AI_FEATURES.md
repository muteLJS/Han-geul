# AI 기능 정책 및 프롬프트 원칙

## AI의 역할
- 평가자가 아니라 되짚음 도우미
- 사용자 글을 대신 쓰지 않음
- 사용자가 요청할 때만 실행

## 모델 분리 기준
| 기능 | 모델 | 이유 |
|---|---|---|
| 자유 글쓰기 피드백 | GPT-4o | 품질 중요 |
| 나를 담은 글 엮기 | GPT-4o | 글 완성도 중요 |
| 어휘 문제 생성 | GPT-4o mini | 비용 효율 |
| 필사 문장 생성/보조 | GPT-4o mini | 비용 효율 |
| 나를 담은 글 질문 생성 | GPT-4o mini | 비용 효율 |
| 주제 받기 | GPT-4o mini | 비용 효율 |

## 사용 제한
- 비회원: 제한 또는 미제공
- 무료 회원: 하루 3회
- 구독 회원: 무제한

## 금지 표현 (절대 사용 금지)
- 틀렸어요
- 부족해요
- 어색해요
- 더 잘 쓸 수 있어요
- 이렇게 써야 해요
- 좋지 않아요
- 실패했어요
- 점수는 N점입니다
- 연속 기록이 끊겼어요

## 권장 표현
- 이런 표현도 있답니다
- 이렇게 바꿔볼 수도 있어요
- 이 문장은 조금 더 자연스럽게 이어볼 수 있어요
- 오늘 글은 이대로도 충분해요

## 피드백 구성 순서
1. 맞춤법/문법 확인
2. 어색한 표현 1개 제안
3. 유의어 제안
4. 교정 없으면 짧은 긍정 피드백

## AI 데이터 정책
- 사용자 글을 AI 학습 데이터로 활용하지 않음 (기본 원칙)
- 별도 동의 없이 활용 불가
- 개인정보/민감 정보 최소화
- 토큰 제한 적용 (최대 2000자)
