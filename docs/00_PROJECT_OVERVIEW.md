# 한-글 프로젝트 개요

## 서비스 정의
- 서비스명: 한-글
- 슬로건: 한 자씩, 한 글을
- 정의: 쓰면서 배우고, 쌓으면서 성장하는 한국어 글쓰기 플랫폼
- 마스코트: 한이 (잉크방울형 비인간 마스코트)
- MVP 플랫폼: 모바일 웹

## 핵심 철학
1. 평가하지 않는다 — 점수, 등급, 실패 표현 없음
2. 과정 중심의 성장 — 결과보다 매일 쓰는 과정
3. 사용자 주도성 — 주제, 공개 범위, 피드백 모두 사용자 선택
4. 결과물 축적 — 글이 도서함과 책으로 남는다

## 페르소나
- 정근우 / 28세 / 직장인 / 실용형 성장 사용자
  - 어휘 연습, 필사, 성장 리포트, AI 피드백 주 사용
- 이젠 / 34세 / 프리랜서 / 자기 언어 탐색 사용자
  - 나를 담은 글, 필사, 비공개 도서함, 책 만들기 주 사용

## 기술 스택
- 프론트: Next.js App Router, React
- DB: MongoDB Atlas
- 인증: NextAuth.js
- 이미지: Cloudinary
- AI: OpenAI API
- 결제: 토스페이먼츠
- 배포: Vercel
- 모니터링: Sentry, Vercel Analytics

## MVP 핵심 루프
쓰기 → 되짚기 → 쌓기 → 책으로 묶기
