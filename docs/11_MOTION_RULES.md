# 한-글 Motion Rules

## 핵심 원칙

한-글의 motion은:
- 조용해야 한다
- 짧아야 한다
- 안정적이어야 한다
- UI를 방해하지 않아야 한다

motion 자체가 주인공이 되면 안 된다.

---

## 기본 등장 애니메이션

기본:
- opacity: 0 → 1
- y: 16px → 0
- duration: 0.45s ~ 0.6s
- easeOut 계열 사용

---

## hover 규칙

- scale 최대 1.02
- duration 0.2s 이하
- 과한 회전 금지
- hover 시 튀는 효과 금지

---

## 버튼 인터랙션

- 살짝 눌리는 느낌만 허용
- scale 0.98 정도만 허용
- bounce 금지

---

## 페이지 전환

- 부드러운 fade 중심
- 강한 슬라이드 전환 금지
- 전체 화면 이동감 최소화

---

## 한이 애니메이션

한이는:
- 작고 부드럽게 움직인다
- 과한 캐릭터 연기 금지
- bounce, elastic 금지

허용:
- 작은 흔들림
- 미세한 y 이동
- opacity 변화

---

## 금지 사항

절대 금지:
- bounce
- elastic
- 과한 rotate
- 큰 scale
- 화면 전체 패럴랙스
- 현기증 나는 모션
- 무한 흔들림
- 과한 particle 효과

---

## 스크롤 인터랙션

- 자연스럽게 이어져야 함
- 갑작스러운 속도 변화 금지
- 모바일 기준 부드럽게 유지
