# 한-글 비주얼 디렉션

## 핵심 분위기
- 조용함
- 절제
- 따뜻함
- 여백감
- 고급스러운 한지 질감
- 경복궁 같은 절제된 한국적 아름다움
- 현대적인 모바일 앱 감성

---

## 디자인 방향

한-글은 일반적인 생산성 앱이나 화려한 감성 앱이 아니다.

중요한 것은:
- 안정감
- 균형감
- 여백
- 타이포
- 질감
- 조용한 인터랙션

기능보다 화면 분위기와 사용자 감정 흐름을 우선한다.

---

## 최우선 시각 요소

우선순위:
1. 한이
2. CTA
3. spacing
4. typography
5. texture
6. 기능 요소

---

## spacing 규칙

- 카드 내부 padding: 최소 20px
- 기본 카드 padding: 24px
- 섹션 간격: 40~56px
- 요소 간격: 12~20px
- 텍스트 줄간격: 1.6 이상

---

## radius 규칙

- 최대 radius: 20px
- 과한 pill 형태 남용 금지
- 작은 요소 radius: 10~14px

---

## shadow 규칙

- opacity 0.06 이하
- spread 최소화
- blur 과도하게 사용 금지
- 떠보이는 UI 금지

예시:
box-shadow: 0 8px 24px rgba(0,0,0,0.06);

---

## 컬러 사용 규칙

- 전체 배경은 항상 한지 계열
- 오방색은 강조용만 사용
- 원색 남용 금지
- 형광 느낌 금지
- 검정은 완전 black 대신 먹색 사용

---

## 타이포 규칙

- 제목은 Serif 계열 중심
- 본문은 가독성 우선
- 작은 모바일 화면에서도 답답하지 않아야 함
- 텍스트보다 여백을 먼저 조정한다

---

## 절대 금지

- glassmorphism
- neumorphism
- 과한 blur
- 과한 gradient
- cartoon 스타일
- 두꺼운 border
- 과한 전통 패턴
- 일본풍 와비사비 느낌
- 과한 bounce motion
- 떠다니는 느낌의 UI
- 사이버펑크 느낌
- flashy animation

---

## 구현 방향

디자인을 재해석하지 않는다.

현재 Figma 기준:
- layout
- spacing
- hierarchy
- proportions

를 최대한 유지한 상태로 구현한다.

새로운 디자인 요소를 임의로 추가하지 않는다.
