## 공통 모달 컴포넌트
| 컴포넌트 | 노출 시점 |
|---|---|
| AuthModal | 비회원이 저장/반응/팔로우 시도 시 |
| SubscribeModal | 구독 제한 기능 접근 시 |
| FeedbackModal | 피드백 받기 클릭 시 |
| SaveModal | 저장 버튼 클릭 시 (책 선택) |
| PublishModal | 공개 범위 설정 시 |

## 반응형 기준
| 구분 | 기준 | 설계 방향 |
|---|---|---|
| 모바일 | ~768px | 하단 탭바, 책장 가로 스크롤, 단일 컬럼 |
| 태블릿 | 768~1024px | 카드 폭 확장, 2열 그리드 |
| 데스크탑 | 1024px~ | 상단 네비, 최대 너비 유지, 그리드 확장 |

## 우선 구현 순서
1. 홈 (F-05)
2. 글쓰기 (F-06) + AI 피드백 모달 (F-07)
3. 로그인 + 비회원 처리
4. 도서함 (F-12)
5. 온보딩 (F-02~F-04)
6. 어휘/필사/나를 담은 글 (F-08~F-10)
7. 나의 책 쓰기 (F-11)
8. 탐색 + 작가 채널 (F-13~F-14)
9. 구독 페이지 + 모달 (F-17)
10. 한이 꾸미기 (F-16)
11. 마이페이지 + 설정 (F-15, F-18)
