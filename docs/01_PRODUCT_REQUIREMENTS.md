# 제품 요구사항 (PRD)

## MVP 포함 기능
- 계정: 카카오·구글·네이버 소셜 로그인, 비회원 임시 작성, 로그인 후 자동 연동
- 홈: 한이 말풍선, 글 쓰기 CTA, 순우리말 원라인, 오방색 기능 버튼, 미션/성장 리포트
- 글쓰기: 제목/본문, 저장, 도움 버튼, AI 피드백, 도서함 저장
- 어휘 연습: 국립국어원 기반 문제, 힌트 3단계
- 필사: 저작권 안전 원문, 난이도 조정, 기록 저장
- 나를 담은 글: AI 질문 기반 자기소개/내면 탐색
- 나의 책 쓰기: 글 불러오기, 챕터 추가, 완성 선언
- 도서함: 책장 UI, 가로 스크롤, 책 두께 3단계
- 탐색: 책과 작가 발견 중심, 랜덤 기본 정렬
- 작가 채널: 프로필, 공개 책·글, 팔로우 버튼
- 구독: 구독 페이지, 제한 기능 접근 시 모달, 토스페이먼츠
- 한이 꾸미기: 기본 미리보기, 포인트 기반 아이템

## MVP 제외 기능
- 댓글 기능
- 외부 공유 실제 구현 (버튼 UI만)
- 푸시 알림
- 광고 모델
- 기부 제도
- 실물책 제작·배송
- iOS·Android 네이티브 앱

## 핵심 사용자 흐름
### 비회원
홈 → 글 쓰기 → 임시저장 → 저장 클릭 시 로그인 유도 → 로그인 후 자동 연동

### 무료 회원
홈 → 글 쓰기 → 저장 → AI 피드백 하루 3회 → 도서함 → 나의 책 추가 → 탐색

### 구독 회원
홈 → 글 쓰기 → AI 피드백 무제한 → 책 만들기 → 공개 설정 → 작가 채널 → 붓 하트/밑줄
