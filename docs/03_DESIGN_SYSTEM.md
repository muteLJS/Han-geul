## 서체
| 영역 | 서체 | 이유 |
|---|---|---|
| 제목 | Noto Serif KR, 나눔명조 | 전통적이지만 단정한 인상 |
| 본문 | Noto Sans KR, KoPub 바탕 | 긴 글 가독성 |
| 한이 말풍선 | Nanum Pen Script | 손글씨 같은 따뜻함 |
| UI 숫자/버튼 | Noto Sans KR | 작은 크기 판독성 |

## 오방색 버튼 시스템
- 배치: 홈에서 원라인 5개 (모바일/데스크탑 동일)
- 크기: 56×56px 원형
- 순서: 청(자유 글쓰기) → 백(어휘) → 적(필사) → 흑(나를 담은 글) → 황(책 쓰기)

## 한이 크기 기준
| 상황 | 크기 |
|---|---|
| 스플래시 / 완료 화면 | 120×168px 이상 |
| 홈 메인 | 64×90px 이상 |
| 글쓰기 플로팅 | 64×90px |
| 모달 내부 | 32×45px |

## 디자인 방향
- 한지 질감, 먹색, 절제된 오방색, 여백, 선 중심
- 경복궁 같은 절제된 전통미 + 현대적 세련됨
- 단순한 도형 나열보다 시각적 디테일과 질감 중시
- 전체 UX는 단순하게, 시각적 완성도는 섬세하게

## 글쓰기 화면 전용
- 중앙 정렬, 최대 너비 700px
- 작성 중 글자 수 카운터 미노출
- 저장 후 또는 리포트에서만 글자 수 확인 가능
